# YOLO 目标检测模型工作站

一个基于 PySide6 和 Ultralytics YOLO 的图形化目标检测应用程序，支持图片、视频和批量目录分析。

## 🚀 功能特性

- **多源输入支持**
  - 🖼️ 单张图片分析
  - 🎞️ 视频实时检测
  - 📁 目录批量扫描

- **实时检测显示**
  - 实时预览检测画面
  - FPS、分辨率和推理时间监控
  - 当前帧目标数量统计

- **历史记录管理**
  - 自动记录所有检测结果
  - 显示类别和置信度
  - 支持导出 CSV 格式

- **可调节参数**
  - 置信度阈值调节 (0-100%)
  - IoU(重叠度) 阈值调节 (0-100%)

## 📋 环境要求

- Python 3.8+
- OpenCV
- PySide6
- Ultralytics YOLO

## 🔧 安装依赖

```bash
pip install opencv-python PySide6 ultralytics
```

## 💻 使用方法

### 启动程序

```bash
python AIapp.py
```

### 操作流程

1. **载入模型**: 点击 `📥 载入权重模型 (.pt)` 选择 YOLO 权重文件
2. **选择源**: 
   - 图片分析：选择单张图像文件
   - 视频检测：选择视频文件
   - 目录扫描：选择包含图像的文件夹
3. **调整参数**: 设置置信度阈值和 IoU 阈值
4. **启动引擎**: 点击 `▶ 启动分析引擎` 开始检测
5. **导出结果**: 点击 `📊 导出全部历史 (CSV)` 保存检测结果

## 🎨 界面说明

### 左侧面板 - 资产管理
- 模型加载按钮
- 输入源选择（图片/视频/目录）
- CSV 导出按钮
- 参数调节滑块
- 启动/停止控制

### 中间面板 - 实时分析预览
- 检测画面实时显示
- 性能监控条（FPS、分辨率、推理时间）

### 右侧面板 - 数据统计
- 当前帧目标数量
- 置信度进度条
- 历史检测清单表格
- 系统日志输出

## 📊 数据格式

导出的 CSV 文件包含以下列：
- 时间：检测时间戳
- 文件名：输入文件名称
- 类别：检测到的目标类别
- 置信度：检测置信度分数

## ⚙️ 默认参数

- 置信度阈值：25%
- IoU 阈值：25%

## 🛠️ 技术栈

- **GUI 框架**: PySide6 (Qt for Python)
- **深度学习框架**: Ultralytics YOLO
- **图像处理**: OpenCV
- **开发语言**: Python 3

## 📝 注意事项

- 确保使用兼容的 YOLOv8/v11 模型文件 (.pt 格式)
- 视频和目录批处理会自动连续执行直到完成
- 所有检测结果会实时添加到历史清单中
