# YOLO目标检测模型 - 智能视觉分析工作站

---

## 项目简介

YoloHistoryWorkstation 是一个基于YOLO深度学习模型的专业目标检测可视化系统，提供实时检测、历史记录管理和数据导出功能。系统采用PySide6构建现代化图形界面，支持多种输入源和灵活的检测参数调节。

---

## 主要功能

### 检测功能
- 单张图片分析：支持常见图片格式(JPG, PNG等)
- 视频实时检测：逐帧处理视频流，实时显示检测结果
- 批量目录扫描：自动处理指定文件夹内所有图片

### 数据管理
- 检测历史记录：自动保存每次检测的详细信息
- 累计统计显示：实时更新检测目标总数
- CSV格式导出：一键导出完整检测历史数据

### 参数调节
- 置信度阈值：0-100%可调，控制检测灵敏度
- IoU阈值：0-100%可调，控制重叠框过滤

### 监控面板
- 实时FPS显示
- 图像分辨率显示
- 处理时间统计
- 置信度分布可视化

## 快速开始

### 启动程序
```bash
python main.py
```

### 基本使用流程

1. **加载模型**
   - 点击"载入权重模型"按钮
   - 选择已下载的.pt格式模型文件

2. **选择输入源**
   - 点击"图片分析"选择单张图片
   - 点击"视频检测"选择视频文件
   - 点击"目录扫描"选择文件夹

3. **调节参数**
   - 滑动置信度阈值滑块(默认25%)
   - 滑动IoU阈值滑块(默认25%)

4. **开始检测**
   - 点击"启动分析引擎"
   - 查看实时检测结果

5. **导出数据**
   - 点击"导出全部历史"
   - 选择保存位置生成CSV文件

## 使用说明

### 输入源类型

| 类型 | 支持格式 | 处理方式 | 说明 |
|------|----------|----------|------|
| 图片 | JPG, PNG, JPEG | 单次处理 | 显示检测结果并记录历史 |
| 视频 | MP4, AVI, MOV | 逐帧处理 | 实时显示，自动停止 |
| 目录 | 文件夹 | 批量处理 | 自动遍历所有图片文件 |

### 参数说明

**置信度阈值**
- 范围：0-100%
- 作用：过滤低置信度的检测结果
- 建议：25-50%为平衡值，100%仅显示最高置信度结果

**IoU阈值**
- 范围：0-100%
- 作用：去除重叠度过高的重复检测框
- 建议：45-60%为常用范围

### 界面元素说明

**左侧面板**
- 资产管理：模型加载和输入源选择
- 引擎参数：检测阈值调节
- 启动引擎：开始/停止检测

**中间面板**
- 实时预览：显示带标注的检测结果
- 监控栏：显示FPS、分辨率、处理时间

**右侧面板**
- 当前帧目标数：实时显示检测数量
- 置信度进度条：可视化平均置信度
- 历史清单：累计显示所有检测记录
- 系统日志：记录操作和检测状态

## 核心功能详解

### 检测流程

1. **图像预处理**
   - 读取输入图像/视频帧
   - 保持原始分辨率

2. **模型推理**
   - 应用置信度和IoU阈值
   - 执行YOLO目标检测

3. **结果处理**
   - 提取检测框坐标
   - 获取类别标签和置信度
   - 绘制标注框

4. **数据存储**
   - 记录检测时间戳
   - 保存文件名和检测结果
   - 更新历史表格

5. **界面更新**
   - 刷新预览图像
   - 更新统计数据
   - 输出性能日志

### 性能优化

- 异步处理：定时器控制视频/目录处理频率
- 内存管理：及时释放图像资源
- 界面响应：主线程保持流畅，避免阻塞

## 导出数据格式

CSV文件包含以下字段：

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| 时间 | string | 检测时间戳 | 14:30:25 |
| 文件名 | string | 源文件名称 | test.jpg |
| 类别 | string | 检测目标类别 | person |
| 置信度 | string | 置信度分数 | 0.92 |

示例数据：
```csv
时间,文件名,类别,置信度
14:30:25,test.jpg,person,0.92
14:30:25,test.jpg,car,0.87
14:30:26,video.mp4,dog,0.76
```

## 常见问题

### 1. 程序无法启动
- 检查Python版本是否为3.7以上
- 确认所有依赖库已正确安装
- 查看控制台错误信息

### 2. 模型加载失败
- 确认模型文件为.pt格式
- 检查模型文件是否完整
- 验证YOLO版本兼容性

### 3. 检测速度慢
- 降低视频分辨率
- 提高处理间隔时间
- 使用更轻量级的模型(如yolov8n)

### 4. CSV导出失败
- 检查是否有检测历史记录
- 确认保存路径有写入权限
- 避免文件名包含特殊字符

### 5. 界面显示异常
- 调整窗口大小刷新显示
- 检查显卡驱动是否更新
- 尝试降低显示分辨率
